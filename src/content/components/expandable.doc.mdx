## Prompt

Build an accordion-style disclosure component. A clickable header reveals/hides body content with a smooth height animation. Include `aria-expanded`, `aria-controls`, and a region role for accessibility. Configurable `defaultOpen`. Use modern CSS grid-row animation (0fr → 1fr) or motion/react. Respect prefers-reduced-motion — instant toggle when set.

## Usage

- Use for FAQs, optional details, "show more" content.
- Pair with chevron icon in header (rotates on open).
- Don't nest deeper than 2 levels — gets confusing.

## Do / Don't

- **Do** keep header content concise (under 12 words).
- **Do** use sentence case for headers.
- **Don't** put critical info inside Expandables — users miss it.
- **Don't** auto-open on page load except for `defaultOpen` props you set.
