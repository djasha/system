# Djasha System

*An AI-native design system. Copy code, copy prompts, copy the whole bundle into any agent.*

**Status:** v1 design spec approved, implementation not yet started.

## What this is

A standalone design system library that will live at [`system.djasha.me`](https://system.djasha.me). Not yet another shadcn clone — the differentiator is **built for AI-agent consumption**: every component ships a natural-language prompt and a full-context bundle (tokens + source + a11y) that any agent (Claude Code, Cursor, v0, Bolt) can paste in one shot to port into a target stack.
